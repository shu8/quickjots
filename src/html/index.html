You might have heard of the data:text/html, <html contenteditable> 'trick' for writing quick notes in your browser, but it's not perfect: most browsers will kill the tab if it's in the background for a while, deleting anything you wrote; you need to remember the awkward URL if you want to use it on other machines; it pastes HTML by default; and so on!

QuickJots lets you jot down and auto-save any quick notes for yourself whilst you're on your computer or phone. It supports both Markdown and simple plain-text notes, and comes with a dark mode and complete privacy!

If you want to help improve QuickJots or get in touch with me, please raise an issue on GitHub, or send me an email.

Features
  • Auto-save Anything you type or paste is saved automatically for you to come back to whenever you want.
  • Works offline Jot down quick notes even when you're offline.
    It's just a website too, so you can use it in any modern browser without needing extra extensions.
  • Privacy Anything you type stays only on your device. You can decide whenever you want to delete notes too.
    QuickJots is open-source on GitHub if you want to see yourself.
  • Markdown support Useful if you want to keep referring back to something and see it in a pretty layout.
    GitHub Flavoured Markdown is supported, so you can use features like tasklists — making it easy to track your progress in a quick to-do list.
    You can also use emojis, insert clickable links, and so much more with Markdown!
  • Plain-text support If you don't want formatting with Markdown, you can use the plain-text editor instead.
  • No registration needed
  • Dark mode!
